# Evidências - Tutorial Git

## 1. Criação e Clonagem de Repositório

O repositório público **POO** foi criado no GitHub e, em seguida, clonado para a máquina local com o comando `git clone`.

![Repositório POO criado no GitHub](imgs/repositorio.png)

![Clonagem do repositório](imgs/clone.png)

## 2. Edição de Conteúdo (commit e push)

O arquivo `README.md` foi editado no editor **nano**, e o `git status` confirmou a modificação. As alterações foram adicionadas, commitadas e enviadas ao repositório remoto com `add`, `commit` e `push`.

![Edição do README.md no nano](imgs/edicao.png)

![Git status mostrando o arquivo modificado](imgs/status.png)

![Fluxo de add, commit e push](imgs/commitpush.png)

## 3. Criação e Alteração de Branch

Uma nova branch chamada `new_modification` foi criada e ativada com `git branch` e `git checkout`.

![Criação e troca de branch](imgs/branch.png)
